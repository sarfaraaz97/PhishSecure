About PhishSecure

PhishSecure is a project that emerged from a final year project “Detection of Phishing Website Using Machine Learning” carried out at Srinivasa Ramanujan Institute of Technology. It aims to help reduce phishing attacks by assisting internet users in authenticating URL links by testing if they are phishing or legitimate. The validation process of a Website URL for phishing or legitimacy has undergone several machine learning models.
